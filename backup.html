Backup Setup Instructions

These are the setup instructions for your automatic nightly backups. Every night, your database and all receipt photos are backed up to secure cloud storage. You need two free accounts — Cloudflare R2 (where the backups are stored) and GitHub (which runs the automated backup job). Both are free and both take about 5 minutes to set up. Once done, send the details to Elliott English (t.elliott.english@gmail.com) and I'll connect everything.

ℹ What This Does

Every night at midnight, an automated job runs that:

  • Backs up your entire database (all submissions, user accounts, store data)
  • Copies all receipt photos to a separate backup location
  • Keeps 30 days of database backups, then automatically removes older ones

If anything ever goes wrong with the main system, we can restore everything from these backups.
